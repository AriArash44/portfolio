# Personal Portfolio Website

This project is a responsive, multilingual portfolio website built with React. It showcases development skills, interactive UI components, and thoughtful cross-browser optimizations (Due to performance differences between Chromium-based and Mozilla-based browsers, a specific component was modified to reduce animation intensity in Chromium environments while maintaining usability and visual integrity.). The site supports both Persian and English languages, both dark and light theme, and is designed to be fast, accessible, and visually engaging.

## Technologies & Tools

- **React:**  
  Core framework used to build dynamic and component-based UI.

- **TypeScript:**  
  Adds static typing for improved developer experience and code reliability.

- **Tailwind CSS & Sass:**  
  Tailwind provides utility-first styling, while Sass enables advanced styling logic and structure.

- **i18n:**  
  Enables full bilingual support for Persian and English, including RTL layout handling.

- **MSW (Mock Service Worker) + TanStack Query:**  
  Used in development mode to simulate API responses and manage data fetching efficiently.

- **Floating UI:**  
  Powers responsive and accessible tooltips and floating elements.

- **Moment-Jalaali:**  
  Provides Shamsi (Jalali) calendar support for Persian date formatting.

- **Vite:**  
  Fast bundler and development server for optimal build performance.

- **GitHub Pages:**  
  Deployment platform for hosting the static build of the site.
